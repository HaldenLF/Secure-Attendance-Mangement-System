# Secure Attendance Management System

## Overview
The Secure Attendance Management System is an Android application designed to streamline the process of attendance tracking. Users can create an account or log in using their credentials or biometrics registered on their phone. The application allows users to check in and out, with the time and date displayed upon checkout.

## Key Features
- **Account Creation**: Users can create an account with their details.
- **Login**: Users can log in using their credentials or via biometrics.
- **Check-In/Check-Out**: Users can check in and out, with the time and date displayed upon checkout.

## Installation
1. Clone the repository:
    ```sh
    git clone https://github.com/HaldenLF/Secure-Attendance-Mangement-System.git
    ```
2. Open the project in Android Studio.
3. Build and run the application on an Android device or emulator.

## Usage
1. **Create an Account**:
    - Open the application and navigate to the account creation screen.
    - Enter your details and create an account.
2. **Login**:
    - Log in using your credentials or via biometrics registered on your phone.
3. **Check-In/Check-Out**:
    - Navigate to the check-in screen.
    - Check in and out as needed. The time and date will be displayed upon checkout.
